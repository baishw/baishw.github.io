技能分享平台

v1.0
将 BingoClaw 技能发布到分享服务器，自动生成精美分享页面， 适配微信朋友圈转发。其他人打开链接即可预览、下载和安装。
分享发布 朋友圈 自动生成页面 API
⬇ 下载 .skill
✨ 功能特性
  • 发布技能 — 上传 .skill 文件，自动解析元信息
  • 分享页面 — 自动生成适配微信朋友圈的精美页面
  • 一键下载 — 访问者点击即可下载安装包
  • 令牌认证 — 发布需安全令牌，防止滥用
  • API 接口 — 支持程序化发布和查询
🔧 安装说明
1. 下载 skill-sharer.skill
2. 解压到 BingoClaw 的 skills/ 目录
3. 配置 config.json（端口、令牌、基础 URL）
4. 启动服务器，上传技能包分享到朋友圈
