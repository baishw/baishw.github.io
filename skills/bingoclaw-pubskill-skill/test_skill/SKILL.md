# 测试技能

这是一个用于验证 PubSkill 功能的测试技能。

## 功能描述

该技能提供一个简单的问候功能，用于测试技能发布流程。

## 使用场景

- 测试技能创建和发布流程
- 验证 API 接口功能

## 输入参数

| 参数 | 类型 | 说明 |
|------|------|------|
| name | string | 用户名称 |

## 输出结果

| 字段 | 类型 | 说明 |
|------|------|------|
| message | string | 问候消息 |

## 示例

输入：
```json
{"name": "张三"}
```

输出：
```json
{"message": "你好，张三！"}
```
