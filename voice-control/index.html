语音助手

v2.0
通过语音直接控制 BingoClaw 的完整方案。说"小文"即可唤醒， 无需打字，完全解放双手。自动消除回声，语音播报回复。
语音唤醒 免提控制 语音播报 回声消除
⬇ 下载 .skill
✨ 功能特性
  • 语音唤醒 — 说"小文"或"小龙虾"即可唤醒助手
  • 连续对话 — 唤醒后直接提问，无需重复唤醒词
  • 语音播报 — AI 回复通过扬声器读出来
  • 回声消除 — 播报时自动屏蔽麦克风，避免回路
  • 多级备选 — 主引擎失效时自动降级到备用方案
🔧 安装说明
1. 下载 voice-control-openclaw.skill
2. 解压到 BingoClaw 的 skills/ 目录
3. 修改 config.json 填入 API 密钥
4. 运行 voice_assistant_wrapper.sh 启动助手
